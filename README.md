# 🃏 High or Low - Statistical Simulator v1.2

Simulador interactivo del juego de cartas "Mayor o Menor" desarrollado en **Python 3.14** con **Tkinter**. El proyecto aplica conceptos de **probabilidad condicional** y **patrones de diseño (MVC)**.

## 📊 Características Estadísticas
- **Cálculo en Tiempo Real:** Probabilidades dinámicas de ganar (P(Mayor), P(Menor), P(Igual)) basadas en el mazo restante.
- **Mazo Finito:** Simulación real de 52 cartas donde cada extracción modifica el espacio muestral.
- **Score Ponderado:** Sistema de puntos basado en el riesgo estadístico de la decisión tomada.
- **Historial de Sesión:** Registro visual de aciertos y errores para análisis de frecuencias.

## 🛠️ Tecnologías
- **Lenguaje:** Python 3.14
- **Interfaz:** Tkinter (Standard Library)
- **Arquitectura:** Model-View-Controller (MVC)
